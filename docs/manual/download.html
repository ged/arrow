Arrow: Downloading

Releases

Though the code isn't yet feature-complete or of final release quality, it is stable enough to experiment and write applications with. You can download the latest preview release from the main project page.

Development Versions

In addition to the above releases, we offer a couple of different ways to get access to the development versions of the code:

Anonymous CVS

To check out a copy from our repository via anonymous CVS:

  • Set your CVSROOT (or cvs -d) to: :pserver:anonymous@www.rubycrafters.com:/www/rubycrafters.com/cvs
  • Check out the 'Arrow' module.

CVS Snapshot

You can also download a snapshot from the repository. This is like checking it out from CVS, but without the ability to update later.

Arrow: Downloading

Releases

Though the code isn't yet feature-complete or of final release quality, it is stable enough to experiment and write applications with. You can download the latest preview release from the main project page.

Development Versions

We recently converted to Subversion for version control, but haven't yet set up anonymous access to the repository. If there's enough interest in anonymous Subversion access, we'll set that up.

In the meantime, you can fetch a snapshot at the RubyForge project page:

http://arrow.RubyForge.org/

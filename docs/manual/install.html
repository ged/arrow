Arrow: Installation

Apache

All development and testing is done using the Apache 1.3.x series webserver on Linux and MacOS X, though you should be able to run it under other platforms and/or Apache 2.0.x. We'd love to hear about you experiences under other configurations, and patches for portability are of course always welcomed.

The Generic Apache Request Library (libapreq)

Arrow requires the libapreq extensions to mod_ruby, which in turn requires that you install the Generic Apache Request Library, or libapreq.

Ruby

Ruby (1.8.1) is strongly urged, as the majority of our work has been done under the development version of ruby.

Arrow has also been tested and seems to work under the 1.8.0 release, but that version of Ruby has some known bugs which are fixed in later versions, so if you're having trouble, try a more recent Ruby.

mod_ruby

The mod_ruby Apache module embeds a Ruby interpreter into Apache, much like mod_perl and mod_python do for their respective interpreters.

Arrow depends on the libapreq extensions to mod_ruby, which at the time of this writing (Jan 22, 2004) is only built in to the development version of mod_ruby. It might also work with the stable release (1.0.7) with the following patch applied.

Ruby Library Dependencies

These are the various ruby libraries you will need to install in addition to the standard distributed ones. Known good versions of each one are in the distribution in the redist/ directory, but you may wish to install newer or customized versions.

  • Ruby-Cache (0.3)
  • FormValidator (0.1.3)
  • HashSlice (1.03)
  • PluginFactory (0.01)

Here also are some standard libraries that are used but may not be included with your ruby distribution (debian, for instance).

  • strscan (0.6.7)

Optionally, you can include the following libraries (with a word on what added functionality they bring in).

  • Test-Unit-Mock (0.03) - to run the tests.
  • Diff (0.4) - used by TestUnitMock.
  • criteria (1.1a) - adds in the ability for DataSources to use this interesting interface to querying.

		<p>We use Subversion for version control. You can see a web view of the
		repository at: </p>

		<div style="margin: 0 2em;">
		  <p>&lt;<a
		  href="http://svn.rubycrafters.com/index.cgi/Arrow">http://svn.rubycrafters.com/index.cgi/Arrow</a>&gt;</p>
		</div>

		<p>You can also check out the latest source read-only like this:</p>

		<p><tt>$ </tt><kbd>svn co svn://svn.RubyCrafters.com/Arrow/trunk Arrow</kbd></p>
<pre>
A  Arrow/.irbrc
A  Arrow/test.rb
A  Arrow/arrow.css
A  Arrow/demo.cfg
<em>[...]</em>
 U Arrow
Checked out revision 192.
</pre>

		<p>This will check out the source into a directory called 'Arrow'.</p>
